Description: The ES3 Specification does not allow the use of reserved words in the dot notation. There is no reason to have this restriction.

The following statement is currently illegal, producing syntax errors. It should be legal. 

if (object.function) {
    object.goto = object.var;
}

Relevant Goal: 1
Relevant Design Principle: 2a, 2b
Relevant ES4 proposal: none
