Deferred proposals

These proposal are no longer being actively considered but could potentially be revived in the future:

  • let expressions
  • types
  • syntax for efficient traits (markm)
  • shallow continuations, a simple and general form for capturing a single function activation (dherman, brendan)
  • byte arrays, based on emerging webapi discussions and implementations
  • Proxies:
    • proxy instanceof, enabling proxies to trap instanceof directly. (tomvc, markm)
    • stopTrapping, enabling direct proxies to be “switched off” (tomvc, markm)
  • Specification Techniques
    • specification language, using Harmony to describe Harmony.
    • Another alternative, translating the ES5 spec. into a JavaScript-based definitional interpreter es5definterp.js.txt
    • narrowing_chapter_16 (markm)
