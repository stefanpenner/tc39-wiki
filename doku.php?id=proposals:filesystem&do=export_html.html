Filesystem Objects

Adobe ExtendScript offers platform independent filesystem objects. The implementation may be worth looking at - it may serve as the basis for a generic implementation:

adobefilefolder.pdf

Currently, the File and Folder objects are defined and implemented for Windows XP, Mac OS X, Lunix/x86, Solaris, HPUX, and AIX.
