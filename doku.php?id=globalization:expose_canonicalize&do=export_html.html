expose CanonicalizeLocaleList and CanonicalizeLanguageTag

This is a proposal to expose CanonicalizeLocaleList and CanonicalizeLanguageTag as a public API

Use cases

  • Client side localization UI and frameworks need to operate on ICU locale tags

Proposed API

 
 
 
/**
 * The CanonicalizeLocaleList operations returns the canonical and case-regularized form of the locales listed in locales argument
 * locales {LocaleList} 
 */
Intl.CanonicalizeLocaleList(locales);
 
/**
 * The CanonicalizeLanguageTag operation returns the canonical and case-regularized form of the locale argument
 * locale {String} 
 */
Intl.CanonicalizeLanguageTag(locale);
