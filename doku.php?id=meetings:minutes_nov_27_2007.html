Public Minutes

Open issues

  • #138 (expression scoped pragmas)
    • ⇒ We remove it
  • #226 (eval versioning)
    • JD: seems necessary and sufficient
    • ⇒ We approve it
  • #271 (optional locator on a unit definition)
    • BE/AWB: We need to at least have comments somewhere (annex, etc) about how web embeddings might interpret a locator string if it looks like a URL
    • ⇒ We accept LH’s proposal
    • ⇒ Action item (LH): make sure the note above about interpretations of locator is written up / tracked
  • #276 (this propagation)
    • Need comments from Waldemar Horwat.
    • ⇒ Deferred until next week
  • #282 (restrictions on “use” and “import”)
    • Current restrictions a minor pain for creative use of namespaces
    • Pragmas don’t have visibility backward
    • It may be that “use strict” and “use decimal” need to be constrained
    • ⇒ We reject the proposal as it stands
    • ⇒ Action item (GH/JD): look for primitives “below” packages to allow packages to be completely desugared, maybe this will provide insight.
  • #283 (aliasing import)
    • ⇒ We kill this feature
  • #305 (magic negative string indexing)
    • ⇒ We kill this proposal
