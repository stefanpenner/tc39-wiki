About this directory

This directory (namespace) contains the individual feature specs under review by the ES4-WG for inclusion in the final ES4 spec.

  • Don’t edit something here unless you expect everyone agrees on it.
    • See the proposals area for ideas-in-progress.
    • Record open issues in http://bugs.ecmascript.org
  • Do edit something here if you see a typo or poor language.
    • The whole point of a wiki is to make group editing as simple as possible.

Feature Specs

Specs go up here after they have stabilized through discussion on es4-discuss.

Specs that have been approved by the WG (they have gone one week past last call without substantive comments) get a smiley face and are to be considered approved for the final language

    • Line terminator normalization
    • Line continuation in string and regular expression literals
    • Triply quoted string literals
  • Pre-defined classes and objects
