Object.observe: Public API Specification

Object.observe

A new function Object.observe(O, callback, accept) is added, which behaves as follows:

  1. If Type(O) is not Object, throw a TypeError exception.
  2. If IsCallable(callback) is false, throw a TypeError exception.
  3. If IsFrozen(callback) is true, throw a TypeError exception.
  4. If accept is undefined
    1. Let acceptList be a new List containing [”add”, “update”, “delete”, “reconfigure”, “setPrototype”, “preventExtensions”]
  5. Else:
    1. Let acceptList be a new List.
    2. If Type(accept) is not Object, throw a TypeError exception.
    3. Let lenValue be the result of Get(accept, “length”)
    4. Let len be ToLength(lenValue).
    5. ReturnIfAbrupt(len).
    6. Let nextIndex be 0.
    7. Repeat while nextIndex < len
      1. Let next be the result of Get(accept, ToString(nextIndex)).
      2. Let nextString be ToString(next).
      3. ReturnIfAbrupt(nextString).
      4. Append nextString to acceptList.
      5. Let nextIndex be nextIndex + 1.
  6. Let notifier be the result of calling [[GetNotifier]], passing O.
  7. Let changeObservers be [[ChangeObservers]] of notifier.
  8. If changeObservers already contains a record whose callback property is the same as callback, then
    1. Perform CreateOwnDataProperty(record, “accept”, acceptList) and
    2. Return O.
  9. Let observerRecord be the result of performing ObjectCreate().
  10. Perform CreateOwnDataProperty(observerRecord, “callback”, callback).
  11. Perform CreateOwnDataProperty(observerRecord, “accept”, acceptList).
  12. Append observerRecord to the end of the changeObservers list.
  13. Let observerCallbacks be [[ObserverCallbacks]].
  14. If observerCallbacks already contains callback, return O.
  15. Append callback to the end of the observerCallbacks list.
  16. Return O.

Object.unobserve

A new function Object.unobserve(O, callback) is added, which behaves as follows:

  1. If Type(O) is not Object, throw a TypeError exception.
  2. If IsCallable(callback) is not true, throw a TypeError exception.
  3. Let notifier be the result of calling [[GetNotifier]], passing O.
  4. Let changeObservers be [[ChangeObservers]] of notifier.
  5. If changeObservers does not contain a record whose callback property is the same as callback, return O.
  6. Remove the record whose callback property is the same as callback from the changeObservers list.
  7. Return O.

Array.observe

A new function Array.observe(o, callback) is added, which is equivalent to 

function(O, callback) {
  return Object.observe(O, callback, ["add", "update", "delete", "splice"]);
}

Array.unobserve

A new function Array.unobserve(o, callback) is added, which is equivalent to 

function(O, callback) {
  return Object.unobserve(O, callback);
}

Object.deliverChangeRecords

A new function Object.deliverChangeRecords(callback) is added, which behaves as follows:

  1. If IsCallable(callback) is not true, throw a TypeError exception.
  2. Call [[DeliverChangeRecords]] with arguments: callback repeatedly until it returns false (no records were pending for delivery and callback no invoked).
  3. Return.

Object.getNotifier

A new function Object.getNotifier(O) is added, which behaves as follows:

  1. If Type(O) is not Object, throw a TypeError exception.
  2. If O is frozen, return null.
  3. Return the result of calling [[GetNotifier]], passing O.
