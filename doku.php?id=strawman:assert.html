Rationale

Create a standardized way for programs to test invariants.

Examples

Asserting: 

assert(x === 12); // throws an AssertionError if false, with a default error message
assert(x === 12, "I think it should be twelve") // throws an AssertionError if false, with the given error message

The AssertionError constructor: 

typeof AssertionError // "function"
AssertionError.length // 1
AssertionError.prototype instanceof Error // true

Spec

  • AssertionError: a constructor whose prototype is an instanceof Error; takes a message (coerced to string) as its first argument
  • assert: a function that takes 1 or 2 arguments and throws a new AssertionError if the first argument is falsey, using the second as the argument to the AssertionError constructor
