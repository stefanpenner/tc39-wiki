Description: ECMAScript 3rd Edition (ES3) does not specify the syntax for date and time literals in 15.9.4.2. This is a source of ambiguity. Implementations have diverged and have come to use many heuristics for parsing date-time formats. This proposal attempts to specify a format for the date and time literals. I have looked at Lars’ proposal for ES4 and tried to keep this compatible.
